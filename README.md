CAREER COMPASS AI

Career Compass AI is an interactive web application that helps students and beginners discover suitable career paths based on their interests and skills.The platform provides career recommendations, learning roadmaps, dashboard analytics, resume scoring, internship suggestions, chatbot assistance, progress tracking, and downloadable career reports.

Features

1. Career Assessment Quiz:

--> Interactive career quiz
--> AI Engineer recommendation
--> Web Developer recommendation
--> Cyber Security recommendation
--> Data Scientist recommendation

2. Learning Roadmaps:

--> Personalized career roadmaps
--> Step-by-step learning guidance
--> Stored using Local Storage

3. Dashboard:

--> Career details display
--> Progress bar tracking
--> Resume readiness score
--> Dashboard reset option

4. Resume Score Calculator:

--> Calculates score based on projects, certifications, and internships
--> Maximum score is 100%

5. Internship Suggestions:

--> Provides internship recommendations according to the selected career

6. Skill Analyzer:

--> Analyzes entered skills and suggests suitable career domains

7. Career Compatibility Chart:

Displays compatibility percentage among

  --> AI Engineer
  --> Web Developer
  --> Cyber Security
  --> Data Scientist

8. Achievement Badges:

Unlock badges such as

   --> Career Explorer
   --> Skill Builder
   --> Career Master

9. Dark Mode:

--> Light/Dark theme switching with saved preferences

10. AI Career Chatbot:

Answers basic questions related to

   --> AI Engineering
   --> Web Development
   --> Cyber Security
   --> Data Science

11. Resume Builder:

Generates a simple resume preview from

   --> Name
   --> Email
   --> Skills

12. Roadmap Completion Tracker:

Tracks learning progress through

  --> Basics
  --> Projects
  --> Internship completion

13. Career Salary Predictor:

--> Predicts expected salary ranges for different careers

14. Productivity Features:

  --> Daily Study Streak
  --> Goal Tracker
  --> Study Timer
  --> Quick Notes
  --> Daily Coding Challenges
  --> Motivational Quotes

15. User Features:

  --> User Profile
  --> Profile Picture Upload
  --> Theme Selector
  --> Achievement Badges
  --> Dark Mode

16. Download Career Report:

--> Generates and downloads a career report

  How To Run

  1. Download or clone the project from [GitHub](https://github.com/dhesika08/pro2)
  2. Open the folder in VS code or Antigravity IDE
  3. Install the Live Server extension
  4. open:
       --> index.html
  5. Right-click and choose:
       --> Open with Live Server
  6. Start exploring Career Compass AI
